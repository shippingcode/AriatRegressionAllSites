********************************* Edit an address FR *********************************************************

Edit address test steps
1. Go to development.ariat.com
2. Close subscription
3. Choose FR
4. Go to Sign in
5. Enter user name & password
6. Press "login" button
7. Click "Edit" in My account (middle navigation)
8. Click "Edit" in Addresses page
9. Change the address nickname
10. Press "Save address"
11. Check if the address nickname was changed in the addresses list
