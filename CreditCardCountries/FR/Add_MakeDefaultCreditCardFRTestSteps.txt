************************************ Add-Make default Credit Card *********************************************

Add - make default credit card test1 
1. Go to development.ariat.com
2. Close subscription
3. Choose FR
4. Go to Sign in
5. Enter user name & password
6. Press login Button
7. Click "ADD" in Payment Information section - My Account page
8. Enter card nickname, name on card, select type card as MASTER CARD, 
enter number, security code, select expiration month & year
9. Press "Apply" button
10. Check the credit card was created in Payment information page
11. Press "Make default card"
12. Check that the card nickname was changed into DEFAULT | card nickname
13. Go to My account
14. Click logout
